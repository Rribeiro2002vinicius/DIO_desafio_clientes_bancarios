# DIO_desafio_clientes_bancarios

## Insights para o atendimento ao cliente no sistema bancário.

Quero que a IA analise comentários sobre o atendimento ao cliente nas seguintes situações:

 1: Bancos sem agências físicas (que são puramente digitais).

 2: Bancos mais tradicionais que focam nas agências físicas.

 Isso será feito parapara identificar as principais causas de reclamação e os pontos positivos, assim conseguindo pontuar as principais melhorias a serem adotadas.


O resultado será usado pela equipe de UX para apoiar os executivos nas decisões para a melhora no sistema de atendimento.


A entrega deve conter:

   1: O estudo deverá considerar 3 áreas de atuação: ChatBot/IA/Assistente Virtual, atendimento humano (presencial ou online) e APP/navegador.
   
  2: Todas as respostas dos tópicos abaixo devem ser separados pelas seguintes faixas de idade, 20-35 anos, 35-55 anos, >55.
  
 3: Os tópicos elogios e reclamações devem ser comparados em uma tabela cuja primeira coluna seja a faixa etária e a primeira linha seja os dois tipos de bancos, além disso cada tópico desses deverá possuir 3 tabelas, uma para cada área de atuação
  
 4: Elogios das 3 áreas respeitando a faixa etária. Os elogios deverão ser comparados numa tabela, a primeira coluna deve possuir a faixa etária e a primeira linha os dois tipos de bancos.
 
 5: Reclamações das 3 áreas respeitando a faixa etária. As reclamações deverão ser comparadas numa tabela, a primeira coluna deve possuir a faixa etária e a primeira linha os dois tipos de bancos
 
 6: Notas de 0 a 10 dos usuários. As notas deverão ser comparadas numa tabela, a primeira linha deve ser a faixa etária e a primeira coluna deve possuir cada uma das 3 áreas discutidas. Nesse caso coloque uma tabela para bancos tradicionais e outra para bancos digitais.
 
 7: No final aponte oque pode ser feito pela equipe de UX e para os desenvolvedores dos aplicativos e responsáveis pelo treinamento de IAs para melhorar o atendimento ao cliente levando em conta a eficiência pelo custo de implementação da mudança.
 

O resultado será considerado bom se os dados forem verídicos e úteis num contexto para apoiarem tomada de decisão.


Contexto: Estou trabalhando com feedbacks de clientes bancários relacionados a experiência dos clientes no Chatbot, aplicativo e atendimento humano. Os dados serão usados para apoiar a tomada de decisão nos seguintes assuntos:

 1: um novo sistema de atendimento online com mais acessibilidade para clientes com idades mais avançadas. 
 
 2: um treinamento de IA para que o chatbot consiga resolver problemas mais complexos ou entender que aquele problema em específico precisa de atendimento humano.
 
 3: entender os pontos positivos e negativos na experiência do usuário em bancos digitais e tradicionais para que possamos implementar no nosso sistema

Dados disponíveis: A base contém comentários sobre os atendimentos, as notas dadas a esse atendimento, idade dos usuários e o tipo de banco em questão.


Critérios de análise: A IA deve classificar os feedbacks por tipo de banco, área de atendimento, faixa etária dos clientes e nota atribuída.


Cuidados e restrições:

Use apenas os dados fornecidos.
Não invente números, causas ou conclusões.
Não exponha dados pessoais ou sensíveis.
Se houver informação insuficiente, indique a limitação.
Use linguagem simples e direta.
